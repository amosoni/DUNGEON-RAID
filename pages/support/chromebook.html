Chromebook Support

Steps to get the best experience on Chromebook and locked-down environments.

Home / Support / Chromebook

Tips

  • Use Chrome in fullscreen (F11) and enable "Use hardware acceleration" in settings.
  • For school devices, try guest mode or a different profile if extensions block iframes.
  • Lower resolution via browser zoom (Ctrl -) for low-end devices to improve FPS.

FAQ

The game doesn't load

Check if a content blocker or extension is stopping third-party iframes. Try a clean profile.

No sound or low FPS

Ensure hardware acceleration is on. Close heavy tabs; reduce resolution via browser zoom.
